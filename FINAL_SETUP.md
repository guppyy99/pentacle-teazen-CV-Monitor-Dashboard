# 🎉 Pentacle 소비자 평가 대시보드 - 최종 완성

## 프로젝트 개요

**티젠 제품을 위한 소비자 평가 통합 모니터링 시스템**

온라인 브랜드 평판, 제품 리뷰, 콘텐츠 인사이트를 수집하고 경쟁사와 비교 분석하는 대시보드입니다.

---

## 🚀 빠른 시작

### 1. 개발 서버 실행
```bash
./start.sh
# 또는
pnpm dev
```

### 2. 로그인
- URL: `http://localhost:3000`
- 아이디: `admin`
- 비밀번호: `admin`

### 📈 브랜드 평판 분석 (`/brand-reputation`)
**비교 분석 중심의 온라인 평판 모니터링**

#### 3단계 프로세스:

**Step 1: 브랜드 제품 입력**
- 티젠 콤부차 (기본)
- 동서 콤부차 (비교)
- 브랜드/제품 추가 버튼으로 n개 추가 가능
- 모니터링 범위 선택 (1~12개월)
- ⏳ 키워드 추천 받기 → 1.5초 로딩

**Step 2: AI 키워드 추천**
- 브랜드별로 8개 키워드 추천
- 티젠: "티젠 콤부차 레몬맛", "티젠 콤부차 다이어트" 등
- 동서: "동서 콤부차", "동서 콤부차 후기" 등
- ⏳ 크롤링 시작 → 2초 로딩

**Step 3: 결과 페이지** (`/brand-reputation/results`)
- 🤖 **AI 종합 분석** (최상단)
  - "AI 요약 보기" 버튼 클릭 → 2초 로딩 후 표시
  - 비교 분석 결과 (티젠 vs 동서)
  - 정량적 성과, 정성적 평가
  - 핵심 인사이트, 개선 제안

- **통계 비교 카드** (티젠 vs 동서 나란히)
  - 총 언급량: 티젠 12,458 vs 동서 9,978
  - 긍정 평가: 티젠 78.4% vs 동서 71.2%
  - 평균 참여도, 주요 채널

- **언급량 추이 차트**
  - ☑️ 브랜드별 체크박스로 토글
  - 티젠/동서 선택 가능
  - 주별 데이터

- **채널별 분포**
  - ☑️ 브랜드 선택 체크박스
  - 티젠: 인스타 42.8% 강세
  - 동서: 블로그 41.3% 강세

- **키워드 분석**
  - 브랜드별 탭
  - 긍정/부정 키워드 분리
  - 긍부정 지수 표시
  - 티젠 78.4% vs 동서 71.2%

**크롤링 대상**
- 네이버 블로그, 카페, 지식인
- 인스타그램, 유튜브
- 커뮤니티 (디시인사이드, 더쿠, 인스티즈)

### 🛍️ 제품 리뷰 비교 (`/product-review`)
**쇼핑몰 리뷰 비교 분석**

**🤖 AI 인사이트** (최상단)
- "AI 인사이트 보기" 버튼 → 2초 로딩
- 전체 제품 분석 결과
- 티젠 콤부차 4.6점, 82.4% 긍정률
- 핵심 강점 및 개선점

**제품 링크 추가**
- URL 입력 → "추가" 버튼
- ⏳ 1.5초 로딩 후 제품 추가
- ✅ URL 유효성 검사
  - 빈칸: "URL을 입력해주세요" + 진동
  - 잘못된 형식: "올바른 URL 형식이 아닙니다" + 진동
- 하드코딩된 10개 제품 풀에서 랜덤 추가

**하드코딩 제품 풀 (10개)**
1. 티젠 콤부차 레몬 - 스마트스토어
2. 동서 콤부차 오리지널 - 쿠팡
3. 설록차 콤부차 - 올리브영
4. 티젠 콤부차 자몽 - 스마트스토어
5. 곰표 콤부차 - G마켓
6. CJ 콤부차 복숭아 - 스마트스토어
7. 풀무원 콤부차 - 쿠팡
8. 롯데 콤부차 - 옥션
9. 오설록 콤부차 - 오설록몰
10. 일화 콤부차 - 쿠팡

**등록된 제품 목록**
- 제품명, 플랫폼 배지
- 평균 평점, 리뷰 수, 긍정률
- 링크 URL 표시
- X 버튼으로 삭제

**4개 분석 탭**
1. **개요**: 제품별 평점, 리뷰 수, 긍정/부정률
2. **감정 분석**: 긍정/부정 막대 그래프
3. **키워드 비교**: 주요 키워드 빈도
4. **리뷰 보기**: 최근 리뷰 목록, 별점, 감정 태그

## 📐 콘텐츠 지수 계산

```
콘텐츠 지수 = 
  (조회수 × 0.7) + 
  (댓글 수 × 0.9) + 
  (좋아요 수 × 0.8) + 
  (신선도 가중치)

신선도 가중치:
- 최근 7일: ×1.2
- 8~14일: ×1.0
- 15~30일: ×0.8
- 31일 이상: ×0.6
```
