# 🎯 소비자 평가 대시보드 가이드

## 프로젝트 개요

티젠 콤부차를 예시로 한 **소비자 평가 통합 모니터링 시스템**입니다.
온라인 브랜드 평판, 제품 리뷰, 콘텐츠 인사이트를 수집하고 비교 분석합니다.

---

## 🎨 주요 특징

✅ **현대적인 UI/UX** - 기존 디자인 유지하면서 소비자 평가에 최적화  
✅ **하드코딩 데이터** - 실제 LLM/크롤링 전 시각화 및 UX 검증  
✅ **비교 분석 중심** - 타사와 자사 데이터를 쉽게 비교  
✅ **완전한 반응형** - 모바일/태블릿/데스크톱 완벽 지원

---

## 📊 주요 기능

### 1. 메인 대시보드 (`/dashboard`)
**통합 모니터링 개요**

- **주요 지표 카드**
  - 총 언급량 (전월 대비 증감)
  - 긍정 평가 비율
  - 평균 참여도 (좋아요+댓글+공유)
  - 주요 채널 (가장 많은 언급)

- **언급량 추이 차트**
  - 일별 / 주별 / 월별 토글
  - 인터랙티브 차트 (마우스 호버 시 상세 정보)

- **Top 5 콘텐츠**
  - 콘텐츠 지수 기준 상위 5개
  - 조회수, 좋아요, 댓글 수 표시
  - 플랫폼별 배지 표시

- **채널별 분포**
  - 파이 차트로 시각화
  - 인스타그램, 블로그, 카페, 유튜브, 커뮤니티 등

- **워드클라우드**
  - 긍정/부정 키워드 분리
  - 키워드 크기로 언급 빈도 표현
  - AI 총평 제공

### 2. 브랜드 평판 모니터링 (`/brand-reputation`)
**기능 A: 온라인 평판 검색 및 분석**

#### 3단계 프로세스:

**Step 1: 브랜드 제품 입력**
- 제품명 입력 (예: 티젠 콤부차)
- 모니터링 범위 선택 (1개월 ~ 1년)

**Step 2: AI 키워드 추천**
- LLM이 추천한 검색 키워드 목록
- 예시: "티젠 콤부차 레몬맛", "티젠 콤부차 다이어트" 등
- 키워드 Accept 후 크롤링 시작

**Step 3: 모니터링 결과**
- 통계 카드 (언급량, 긍정률, 참여도, 주요 채널)
- 언급량 추이 차트 (일/주/월 토글)
- Top 5 인터랙션 콘텐츠
- 채널별 분포 (파이 차트)
- 워드클라우드 (긍정/부정)
- AI 총평

**크롤링 대상 플랫폼:**
- 네이버 블로그, 카페, 지식인
- 인스타그램
- 유튜브
- 커뮤니티 (디시인사이드, 더쿠, 인스티즈)

### 3. 제품 리뷰 비교 (`/product-review`)
**기능 B: 쇼핑몰 리뷰 비교 분석**

#### 주요 기능:

**제품 링크 추가**
- 스마트스토어, 쿠팡, 지마켓, 올리브영 등
- n개 제품 동시 비교 가능

**4개 분석 탭:**

1. **개요 탭**
   - 평균 평점 (별점)
   - 총 리뷰 수
   - 긍정/부정 비율

2. **감정 분석 탭**
   - 긍정/부정 리뷰 비율 비교
   - 막대 그래프로 시각화

3. **키워드 비교 탭**
   - 주요 키워드 빈도 비교
   - 예: "맛있다", "상쾌", "다이어트", "가격" 등

4. **리뷰 보기 탭**
   - 최근 리뷰 목록
   - 별점, 날짜, 작성자
   - 긍정/부정 감정 태그
   - 날짜 필터링 (1주일/1달/3개월/1년)

**수집 데이터:**
- 별점
- 날짜
- 구매 제품
- 리뷰 내용
- LLM 감정 분석 결과

### 4. 콘텐츠 평판 분석 (`/content-reputation`)
**기능 C: 콘텐츠별 인사이트 비교**

#### 주요 기능:

**콘텐츠 추가**
- 유튜브 링크
- 인스타그램 링크
- n개 콘텐츠 동시 비교

**3개 분석 탭:**

1. **정량 분석**
   - 조회수, 좋아요, 댓글, 공유 수 비교
   - 막대 그래프
   - 시간별 누적 참여도 추이 (라인 차트)

2. **정성 분석**
   - 긍정/부정 비율 (LLM 분석)
   - 주요 반응 키워드
   - 대표 댓글 표시
   - 플랫폼별 아이콘 표시

3. **AI 총평**
   - 정량적 성과 요약
   - 정성적 평가
   - 개선 제안
   - 종합 분석 리포트

**수집 인사이트:**
- 조회수
- 좋아요 수
- 댓글 수
- 공유 수
- 참여율 (Engagement Rate)

## 📐 콘텐츠 지수 계산

인터랙션이 많은 콘텐츠를 선별하는 지수:

```
콘텐츠 지수 = 
  (조회수 × 0.7) + 
  (댓글 수 × 0.9) + 
  (좋아요 수 × 0.8) + 
  (신선도 가중치)
```

**신선도 가중치:**
- 최근 7일: ×1.2
- 8~14일: ×1.0
- 15~30일: ×0.8
- 31일 이상: ×0.6

- 부정 키워드 증가: 즉시 대응 필요

## 🎓 용어 설명

**콘텐츠 지수**: 인터랙션을 종합한 콘텐츠 영향력 지표  
**참여율**: (좋아요+댓글+공유) / 조회수 × 100  
**긍정률**: 긍정 리뷰 / 전체 리뷰 × 100

**워드클라우드**: 키워드 빈도를 시각적으로 표현한 그래프
